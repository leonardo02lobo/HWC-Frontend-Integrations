# 🚀 Hello World Cup - Frontend

Bienvenido al repositorio oficial del frontend para **Hello World Cup**. Esta aplicación web sirve como el punto central de información, registro y exhibición para nuestra comunidad y competencia de programación.

El proyecto está construido con un enfoque en el rendimiento y la experiencia de usuario, utilizando una arquitectura moderna de sitio estático (SSG) para asegurar tiempos de carga rápidos.

## 📋 Características del Proyecto

La aplicación consta de tres secciones principales diseñadas para guiar a los participantes y mostrar el talento de la comunidad:

### 1. 🏠 Inicio / Competencia (Landing Page)
* **Información General:** Detalles clave sobre el hackathon (fechas, ubicación, modalidad).
* **Reglas y Categorías:** Explicación clara de los tracks de la competencia.
* **Premios:** Visualización jerárquica de los premios para los ganadores.
* **Call to Action (CTA):** Botones directos para el registro de participantes.

### 2. 🎤 Charlas (Talks)
* **Agenda:** Cronograma detallado de las conferencias y talleres.
* **Speakers:** Tarjetas informativas con biografía, foto y redes sociales de los ponentes invitados.

### 3. 💡 Proyectos de la Comunidad
* **Galería de Proyectos:** Un espacio para exhibir desarrollos previos o actuales de la comunidad.
* **Showcase:** Cada proyecto incluye título, equipo, repositorio y demo en vivo.
* **Involucramiento:** Sección para invitar a otros desarrolladores a enviar sus propias creaciones.

## 🛠️ Stack Tecnológico

Este proyecto utiliza tecnologías modernas para garantizar un desarrollo ágil y un producto final optimizado:

* **[Astro](https://astro.build/):** Framework web principal para generar un sitio estático ultra rápido (Island Architecture).
* **[TypeScript](https://www.typescriptlang.org/):** Para un código robusto, tipado y escalable.
* **[Tailwind CSS](https://tailwindcss.com/):** Framework de utilidades para un diseño rápido y responsivo.
* **[Tailwind CSS Animations](https://github.com/jamiebuilds/tailwindcss-animate):** Plugin para agregar animaciones fluidas y mejorar la interactividad de la UI.
